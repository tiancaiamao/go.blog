今天有同事问起，如何并发执行任务，同时保持让结果有序。

我记得以前写过一段代码，想起来很精妙，应该 share 出来。

type task struct {
    sync.WaitGroup
}
func main() {
    for t := range <-task {
        sendToWorker(ch, t)
        sendToKeepOrder(fifo, t)
    }

    for t := range <-fifo {
        t.wait()
    }
}
func worker(ch) {
    for t := range ch {
        t.Done()
    }
}

同时把任务往两个队列里面扔，一个用于实现并发，另一个用于实现先进先出。同时用一个 WaitGroup 来保序。
